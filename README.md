# eCommerce Automation with Selenium+POM+TestNG+Listener+ExtendReport

This project is a Selenium-based automation testing framework for an eCommerce website. It utilizes TestNG for test management, the Page Object Model (POM) design pattern for better code organization, a custom listener for reporting, and Extent Reports for generating detailed test reports.

## Table of Contents
- [Features](#features)
- [Prerequisites](#prerequisites)
- [Setup Instructions](#setup-instructions)

## Features
- Automated testing of eCommerce functionalities
- Maven as build management tool
- Organized code using Page Object Model (POM)
- TestNG for test management and execution
- Custom listener for test event handling
- Extent Reports for detailed reporting

## Prerequisites
- Java Development Kit (JDK) 8 or higher
- Apache Maven
- An IDE (like IntelliJ IDEA or Eclipse)
- Selenium WebDriver
- TestNG
- Extent Reports
